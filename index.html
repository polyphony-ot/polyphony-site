  • Cross-platform
    The core of polyphony is written in pure C - making it portable to any platform.
  • Simple
    A simple API makes it easy to add real-time text editing to virtually any app or website.
  • Open Source
    All of polyphony is open source under Apache 2, and can be used in commercial projects.

Cross-platform Collaboration

Many real-time text editors focus on the browser, making them difficult to use anywhere else. Polyphony provides a clean API for adding real-time collaboration directly into your apps - whether they're built for the browser, for a device, or for the desktop.

Contributing

Polyphony is looking for contributors! We pride ourselves on the code being clean, extensively documented, and well-tested. If you're interested in learning how a real-time text editor works or want to contribute new features, please let us know on GitHub.
